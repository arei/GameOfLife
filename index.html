Welcome to Conway's Game of Life.
To run this application simply press the Start button or the Step above. Start will run the game without interruption, whereas Step will only run one single tick of the game engine. You can use Reset to regenerate a new randomly seeded board at any time, even while running. Stop will stop a currently running game.
A number of options about the game can also be set while the games is not running. The include Cell Size which indicates how big a single cell should be, Initial Seed which determines the percentage of any single cell being on or off when the board is randomly generated, Repeating which, when on, causes the game to automatically generate a new board when the prior game comes to it's conclusion, and Throttling which will limit the number of game itterations that may occur in a single second.
